We have all used flash cards at some point in our lives, whether to study for a final exam or write down your grocery list. However, flash cards can inconveniently disappear behind your couch and become unorganized very quickly if you're not careful. Flash Stack, a digital flash card web application, aims to prevent these problems and give the user a way to store flash cards on an online account for future usage.

Flash Stack is a full-stack web application, consisting of JavaScript technologies on both the client side and the server side. Front end technologies used in the app's construction include AJAX, jQuery, SASS and Boostrap, while server side technologies include Node.js, Express and MongoDB. I designed and built this application while studying web development at Refactor U. You can find and use this web application at: FLASHSTACKAPP.COM
